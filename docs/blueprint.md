# **App Name**: ShopStack

## Core Features:

- Product Catalog: Display products with filtering, sorting, and search functionalities.
- Shopping Cart: Add, remove, and update quantities of products in the shopping cart.
- Checkout: Capture user information (Name, Email, Address, Mobile Number) and payment details using a mock/sandbox payment gateway.
- User Account Management: User registration, login, and password reset functionality.
- Order Management: Create and fulfill orders and track their status.
- Inventory Management: Update stock levels automatically upon order placement.
- Order Notifications: Send email confirmations to users upon order placement (using fake API keys if necessary).

## Style Guidelines:

- Primary color: A muted teal (#77D8D3) to convey trust and stability.
- Background color: A light gray (#F0F0F0), desaturated teal to provide a clean and neutral backdrop.
- Accent color: A soft lime green (#BCE29E) to highlight key interactive elements and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern yet friendly appearance.
- Use clean and simple line icons to represent product categories and actions.
- A grid-based layout with a focus on clear hierarchy and readability.
- Subtle transitions and animations for a smooth user experience when navigating the platform.
